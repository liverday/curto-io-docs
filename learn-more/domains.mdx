Para vincular o domínio da sua marca com o Curto, você vai precisar configurar domínio desejado nas suas configurações no Curto e adicionar os registros DNS em seu provedor.

## Pré-requisitos

- Um domínio/subdomínio que não esteja sendo usado.
- Acesso à configurações de DNS do seu provedor de domínio.

## Adicionando o domínio customizado no Curto

Navegue até a página de [Domínios](https://curto.io/domains) e clique em "Adicionar domínio". Envie o domínio desejado sem o prefixo `https://www`, como por exemplo, `exemplo.com.br`. 

![Adding Domain GIF](/images/domains.gif)

## Adicione os Registros DNS

Navegue até as configurações de DNS do seu provedor. Adicione um novo registro DNS com os seguintes valores

Se for um domínio, como `exemplo.com.br`, adicione uma [Entrada do Tipo A](https://www.cloudflare.com/pt-br/learning/dns/dns-records/dns-a-record):

```
Tipo    Nome    Valor
A       @       76.76.21.21
```

Se for um subdomínio, como `subdominio.exemplo.com.br`, adicione uma [Entrada do Tipo CNAME](https://www.cloudflare.com/pt-br/learning/dns/dns-records/dns-cname-record):

```
Tipo      Nome          Valor
CNAME     subdominio    cname.vercel-dns.com.
```
